Propriedades CSS:

:focus

A propriedade :focus é um seletor de classe que é muito utilizada em formulários, quando o usuario seleciona o elemento clicando nele com o mouse a frase dentro da caixa muda de cor, exemplo de como pode ser feito isso na linguagem CSS:

.nome:focus { color: rgb(152,89,212) }



efeito de animação

Os efeitos de animação são ativados quando passa o mouse sobre o botão que está com a animação, segue um exemplo de como pode ser feita a animação na linguagem CSS:
Nessa animação ao passar o mousse sobre o botão ele vira e mostra outra frase escrita, segue abaixo o exemplo de como fazer essa animação acontecer:

Primeiro você vai criar uma class, a minha ta com o nome .efeito, nessa class você vai definir como você quer a cor, tamanho, tamanho da frase, do botão, etc..
veja um exemplo:

.efeito { border:none;
color: blueviolet;
padding: 20px;
margin: 10px;
font-size: 24px;
line-height: 24px;
border-radius: 18px;
position: relative;
box-sizing: border-box;
cursor: pointer;
transition: all 400ms ease;
}


css do efeito:

.efeito1 {
background-color: rgb(65, 7, 120);
transform-style: preserve-3d;
}
.efeito1:after {
background: rgb(93, 19, 162);
border-radius: 18px;
box-sizing: border-box;
content: "mudou";
transform-origin: left bottom;
transform: rotateX(90deg);
top: 100%;
left: 8px;
width: 100%;
padding: 20px;
position:absolute;
}
.efeito1:hover {
transform-origin:center bottom;
transform:rotateX(-90deg) translate(100%);
}



Background-imagem

A Propriedade Background-imagem define o plano de fundo de um elemento, tendo uma foto de fundo, pode ter também uma caixa com borda, caixa colorida com o texto em destaque, caixa de preenchimento ou caixa de contéudo.
A linguagem css background-imagem é feita da seguinte forma:


.image{
font-size:2.9em;
color:rgb(197,240,255);
background-image:none;
}
div{
background-image:
url(imagem.png)
}
#background-image:
url(imagem.png);

o elemento ficará dessa forma
