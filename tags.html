Descobrindo tags

<sup> </sup>

Essa tag HTML é utilizada para deixar um texto sobrescrito.

Exemplo de utilização: <sup> essa parte do texto esta sobrescrita </sup>

Resultado: texto sobrescrito


<input> </input>

Essa tag permite que tenha um formulário na página onde o usuario pode preenchê-la.

Exemplo de utilização: <input> type="text" name"nome" placeholder="digite seu nome"> campo para formulario </input>

Resultado:


<q> </q>

O elemento HTML <q> indica que o texto dentro da tag é uma pequena citação. Este elemento destina-se a citações curtas que não requerem marcações de parágrafo.

Exemplo de utilização: <q> "essa parte do texto fica citada" </q>

Resultado: Texto citado


<dfn> </dfn>

A tag <dfn> é um termo de definição, onde a letra fica escrita de forma diferente das outras.

Exemplo de utilização: <dfn> a palavra esta escrita com um termo de definição </dfn>

Resultado: Texto de definição
